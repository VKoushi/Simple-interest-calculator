# Simple Interest Calculator
A simple script to calculate the interest earned over a period of time.
## Description
This project calculates simple interest based on the principal amount, the annual interest rate, and the time in years.
## Formula
The calculation is based on the standard formula:
Interest = (Principal * Rate * Time) / 100
## How to Use
1. Input the **Principal** amount.
2. Input the **Annual Interest Rate**.
3. Input the **Time** period in years.
4. The tool will output the total interest earned.
